KANSAS PRAIRIE WEBWORKS — IMAGES REQUIRED
==========================================

Place the following image files in this folder before launch:

REQUIRED FILES:
- kpw-logo-badge.png        KPW badge/logo (used in hero badge and nav)
- kpw-hero-storm.jpg        Kansas storm landscape (hero background, index.html)
- kpw-ad-windmill-sunset.jpg  Windmill at sunset (ad/promotional use)
- kpw-ad-google-profile.jpg   Google Profile service page image
- kpw-prairie-wide.jpg      Wide prairie landscape (about.html hero)
- dogebeats-screenshot.jpg  Screenshot of dogebeats.com (portfolio + service-web-app)

NOTES:
- All images should be optimized for web (WebP preferred, JPG/PNG acceptable)
- Recommended sizes:
    Hero images: 1920x1080px minimum
    Service images: 800x600px minimum
    Logo badge: 200x200px (square)
    Portfolio screenshots: 800x600px minimum
- Keep file sizes under 200KB each for performance
- All images are referenced from /images/ in HTML files

PLACEHOLDERS:
If images are not yet available, the site will use CSS gradient fallbacks.
Add images here and the site will automatically use them.
